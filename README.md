# Pytorch_RL
This project aims to reproduce the results of several RL algorithms. Recently, I will start from DQN. 

If you are looking for awesome_RL resources, you can click [here]()

# Requirements

- Python >=3.8
- Gym 
- Pytorch >=0.4
